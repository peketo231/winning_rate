# winning_rate

## サービス概要
毎年発売されるサッカーゲーム(ウイニングイレブン)を作品ごとの勝率で比較するサービスです。

### 登場人物
- エンドユーザー
  - ウイニングイレブンユーザー
- 管理者
  - 新作が発売されたら更新する人

### ユーザーが抱える課題
現在の勝率が前作と比較してどのくらい変動したのか一目見ただけでわかるようにしたい。

### 解決方法
前作と現在の勝率をグラフで表示することによって、どのくらいゲームが上達したのかがわかる。

### プロダクト
勝率の変動を視覚的に表現するWebアプリケーション

### マーケット
ウイニングイレブンユーザー
